Blockers before promoting v1.0 publicly
----------------------------------------
  [ ] Code signing cert
      - Free path: SignPath Foundation (OSS program). Apply at
          https://signpath.org/apply
        Requires the project be genuine MIT/Apache OSS - id4pii qualifies.
      - Cheap paid: Azure Trusted Signing ~$10/month, no hardware token.
      - Traditional: EV cert (~$300-600/yr) from SSL.com / Sectigo for
        zero-warning SmartScreen from day one.
      - After provisioning, set repo secret:
          ID4PII_INSTALLER_SIGNTOOL = signtool sign /fd SHA256 /tr <ts-url> /td SHA256 $f
        and re-cut release. .iss + release.yml already wired.
